
# Book Recommendation System using Machine Learning

This project builds a simple book recommendation system using KNN (Nearest Neighbors).
The system recommends similar books based on ratings, language, and review counts.

## Features
- Data exploration and visualization
- Feature engineering
- MinMax scaling
- KNN based recommendation system

## Dataset
Dataset file: books.csv

Example source:
https://www.kaggle.com/datasets/jealousleopard/goodreadsbooks

Place dataset inside **data/books.csv**

## Run

pip install -r requirements.txt
python main.py
