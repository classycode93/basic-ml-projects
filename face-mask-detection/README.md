
# Face Mask Detection using Deep Learning

This project detects whether a person is wearing a mask or not using a CNN model.

## Features
- Face detection using OpenCV DNN
- CNN model for mask classification
- Image preprocessing and augmentation
- Real-time mask detection pipeline

## Dataset

Face mask dataset (images + annotations).

Example sources:
https://www.kaggle.com/datasets/andrewmvd/face-mask-detection

Place images inside **data/images/**.

## Run

pip install -r requirements.txt
python main.py
